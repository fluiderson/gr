# System SDKs

Collection crate for system module SDKs.

## Overview

The `cyberware-system-sdks` crate re-exports individual system SDK crates behind feature flags.

## Features

- `directory`: enables `cyberware-system-sdk-directory`
- `directory_grpc`: enables `directory` and the gRPC feature of the directory SDK
